Belajar HTML

HTML (HyperText Markup Language) adalah bahasa standar untuk membuat struktur dan konten halaman web, menggunakan "tag" untuk memberi tahu browser cara menampilkan teks, gambar, link, dan elemen lainnya, menjadi kerangka dasar hampir semua situs di internet. Meskipun sering disebut bahasa pemrograman, HTML sebenarnya adalah bahasa markup yang mendefinisikan susunan konten, bukan logika dinamis, dan merupakan fondasi utama pengembangan web.
